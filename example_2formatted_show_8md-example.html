Formatted output as a table.

Formatted output as a table.

Note
The output contains 2 columns - the policy key and the value. The output width is the length of the longest line + 3 empty characters for each of the columns.
Parameters
policiesA dictionary of lists containing policy keys and values. @type type Object type is machine or user.
nameObject Name.
Returns
Formatted printing of the policies.
